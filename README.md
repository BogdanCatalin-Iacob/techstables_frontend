## Development
-   ### Making a clone
    1. Login to Github and locate [Techstables Frontend](https://github.com/BogdanCatalin-Iacob/techstables_frontend.git)
    2. Click `<> Code` button adn then choose a preffered cloning method
    3. To clone the repository using HTTPS, copy the link under the "HTTPS" tab.
    You can, also, choose to open with "Github Desktop", or download the zip file.
    4. Open the terminal on your computer.
    5. Navigate to the location where you want to create the clone.
    6. Type `git clone`, and the paste the URL copied. `($ git clone https://github.com/BogdanCatalin-Iacob/techstables_frontend.git)`
    7. Press `Enter` on keyboard to create the clone of the repository.
