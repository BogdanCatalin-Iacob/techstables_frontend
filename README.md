## Blog website (ReactJS + REST API)
A modern, responsive bllog application built with ReactJs and powered by a REST API.
This project demonstrates real-world frontend develompent skills, including API integration, state management, routing, and clean UI design.

## Project goals
-   Build a clean and responsive blog frontend using ReactJS
-   Integrate a REST API for fetching and managing blog posts
-   Implement core blog features(list, view, create, edit, delete posts)
-   Apply scalable state management using React hooks
-   Maintain clean, modular, and reusable code
-   Ensure good UX with loading states, error handling, and accessibility
-   Deploy the application to a modern hosting platform (Heroku)
-   Document the project clearly for easy setup

## Features
-   View all blog posts
-   Read a single post with full content
-   Create new posts
-   Edit existing posts
-   Delete posts
-   User authentication (JWT)
-   Responsive UI for mobileand desktop
-   Error + loading state handling
-    Clean component structure

## Technologies
-   ReactJS
-   React Router
-   CSS modules
-   Axios
-   Heroku

## Development
-   ### Making a clone
    1. Login to Github and locate [Techstables Frontend](https://github.com/BogdanCatalin-Iacob/techstables_frontend.git)
    2. Click `<> Code` button adn then choose a preffered cloning method
    3. To clone the repository using HTTPS, copy the link under the "HTTPS" tab.
    You can, also, choose to open with "Github Desktop", or download the zip file.
    4. Open the terminal on your computer.
    5. Navigate to the location where you want to create the clone.
    6. Type `git clone`, and the paste the URL copied. `($ git clone https://github.com/BogdanCatalin-Iacob/techstables_frontend.git)`
    7. Press `Enter` on keyboard to create the clone of the repository.
